# GPUs

## GPU Memory

- SRAM - faster but larger
- DRAM - smaller but slower

A small amount of faster memory is placed next to compute unit with progressively larger and larger slower memory further away.

- GPU memory consists of:
- 1. Device memory
- 2. L2 cache
  3. Distributed Shared Memory
  4. L1 cache and shared memory
  5. Register file
